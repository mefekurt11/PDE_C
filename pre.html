The Challenge of Modern Imaging

  • Data Burden: High-resolution formats strain storage and transmission bandwidth.
  • Conventional Limits: JPEG (DCT) and JPEG2000 (DWT) produce non-semantic distortions.
  • Second-Generation Coding: Exploiting semantic edges and homogeneous diffusion for ultra-high compression.
  • Implementation: High-performance C11 codec using JBIG and Multigrid solvers.

Method I: Discrete Cosine Transform

The DCT converts spatial data into frequency components, concentrating signal energy in low-frequency coefficients.

F(u,v)=α(u)α(v)∑∑f(x,y)cos[...]cos[...]

Implementation uses matrix-based `im2col` for fast block processing in MATLAB.

Method II: Discrete Wavelet Transform

  • Multi-Level Decomposition: Analyzing signal in both space and frequency using Filter Banks.
  • Subbands: Separating Approximation (LL) from Details (LH, HL, HH).
  • Deadzone Quantization: Discarding unimportant detail coefficients.
  • Artifacts: Ringing or Gibbs phenomena near high-contrast edges.

Implementation details of the Edge-Based C Codec

PDE-Based Theory: Stability Fixes

  • Laplace Equation: Reconstructing the background via $\Delta u = 0$.
  • The Dirichlet Problem: Solving the smoothest interpolation between "known pixels".
  • Background Anchoring (The Fix): Regular sampling and block variance tests to avoid color drift in uniform regions.
  • Mask Composition: Semantic Edges + Image Borders + Interior Anchors.

Constraint Mask

Stores 1-D color signal and binary mask. Mask is compressed using JBIG for optimal bitrates.

Algorithmic Pipeline (C11 Implementation)

Encoder Stage

  • Gaussian Presmoothing: Separable 2D kernels for noise reduction.
  • Marr-Hildreth Variant: Edge detection via Laplacian magnitude thresholding.
  • Max-Lloyd Quantization: Iterative optimization of the 1-D color signal levels.
  • JBIG Encoding: Bit-packing the binary constraint mask.

Decoder Stage

  • JBIG Decoding: Reconstructing the 2D grid mask.
  • Signal Scattering: Mapping 1-D quantized samples back to Dirichlet points.
  • Multigrid Solver: W-cycle iterations with restriction/relaxation for convergence.
  • Independent Channels: Parallel reconstruction of R, G, and B planes.

Multigrid Solver: W-Cycle

Direct relaxation (Gauss-Seidel) is slow to propagate global color. We implement a Multigrid hierarchy.

W-Cycle: Pre-smooth → Restrict → Coarse Solve (Recursive) → Prolongate → Post-smooth.

Achieves faster convergence and eliminates local artifacts by solving the error on coarser resolutions.

Conclusion & Future Work

  • Key Takeaway: Geometry-based coding is superior for cartoon-like content where edges define semantic meaning.
  • PDE Strength: Homogeneous diffusion with background anchors solves the color-drift problem.
  • Optimization: JBIG and Multigrid solvers enable real-time performance on a CPU.
  • Future Improvements: Adding 1-D signal entropy coding (zstd/Huffman) and improving prolongation logic.
